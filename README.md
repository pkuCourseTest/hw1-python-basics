# 作业1：Python 基础练习

## 📋 任务

完成 `src/solution.py` 中的三个函数：

| 函数 | 说明 | 分值 |
|------|------|------|
| `reverse_string(s)` | 反转字符串 | 30分 |
| `count_vowels(s)` | 统计元音字母个数 | 30分 |
| `find_max(nums)` | 找最大值（不用max） | 40分 |

## 🚀 如何开始

1. Clone 你的仓库
2. 编辑 `src/solution.py`
3. 本地测试：`pip install pytest && pytest tests/ -v`
4. 提交：`git add . && git commit -m "完成作业" && git push`
5. Push 后会自动评分，去 Actions 标签页查看结果

## ⚠️ 注意

- 不要修改 `tests/` 目录下的文件
- 不要修改函数签名
- 截止日期：请查看 Classroom 页面
